Vditor Markdown Editor Assets
==============================

This directory contains the Vditor editor files for the Notes tab.

Files needed:
1. vditor.min.js - Main JavaScript file
2. vditor.min.css - Stylesheet
3. editor.html - HTML wrapper with configuration

Downloading assets:
-------------------

From unpkg CDN:
curl -L -o vditor.min.js "https://unpkg.com/vditor/dist/index.min.js"
curl -L -o vditor.min.css "https://unpkg.com/vditor/dist/index.css"

Or from jsDelivr:
curl -L -o vditor.min.js "https://cdn.jsdelivr.net/npm/vditor/dist/index.min.js"
curl -L -o vditor.min.css "https://cdn.jsdelivr.net/npm/vditor/dist/index.css"
